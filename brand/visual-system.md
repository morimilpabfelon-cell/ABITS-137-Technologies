# Visual System

## Direction

The visual identity should feel institutional, technical, minimal, and premium.

The reference direction is a light industrial interface: soft gray background, large white surface, black typography, cyan signal accent, modular data fields.

## Color System

```txt
Ink Black:       #07090D
Surface White:   #F4F4EF
Page Gray:       #D8D8D4
Cyan Signal:     #5CECFF
Soft Line:       rgba(7, 9, 13, 0.09)
```

## Logo Hierarchy

Preferred vertical lockup:

```txt
ABITS-137
TECHNOLOGIES
```

`ABITS-137` must dominate. `TECHNOLOGIES` should be smaller, spaced, and stable.

## Design Rules

- Prioritize legibility.
- Avoid excessive glitch.
- Use motion only as system behavior, not decoration.
- Use cyan as signal, not as the main brand color.
- Keep typography strong and black.
- Do not make the brand look like a gaming intro.
